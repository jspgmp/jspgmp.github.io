강의 2주차가 거의 끝이 났습니다.

지금 진도로는 HTML, CSS, JS 기초 단계를 공부했으며, C언어도 살짝 공부하였습니다.

먼저 자바스크립트 함수부터 정리를 해볼까 합니다.

  1. forEach 과 화살표 함수

forEach는 배열에 포함된 각각의 인자에 대해서 가해줄 먼가를 정의하는 것입니다.

const array = [1, 2, 3, 4, 5]

array.forEach(number=> {

​ console.log(number);

})

->array 배열의 인자에 대해서 각각 number 함수를 대입하라

=> 는 함수를 뜻하며 이렇게 표현합니다.

number => {console.log(number)};

=

function (number) {console.log(number);}

function 문구가 날아가고 number를 감싸던 소괄호가 =>로 바뀌었습니다.

만약 array.push(1) 을 넣으면 배열 마지막에 1이 들어가게 됩니다.

  1. map 함수

forEach와 map 의 차이는 ~입니다.

가. forEach를 사용한 경우

const array = [1,2,3,4,5];

const squared =[];

array.forEach(n =>{

spquare.push(n*n);

});

console.log(squared);

나. map을 사용한 경우

const array = [1,2,3,4,5];

const squared = array.map(n=>n*n); //n을 받아와서 n x n을 반환한다!

console.log(squared);

map 대신에 그냥 forEach를 써도 될것같지만, forEach를 사용하면 undefined 로 나옵니다.

map은 리턴값을 저장하지만 forEach는 저장하지 못한다고 합니다.

forEach 와 map 둘 다 배열[]에서만 사용 가능합니다.
